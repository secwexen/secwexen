# Protocol Behavior Map Diagram

This diagram illustrates the expected vs. anomalous behaviors of common network protocols.  
It is designed to support threat detection, SOC workflows, and forensic analysis.

---

## Diagram Overview

The map includes:

- **Protocol Categories**: Application (HTTP, FTP), Transport (TCP, UDP), Network (ICMP, DNS)
- **Normal Behavior**: Typical usage patterns, port ranges, authentication flows
- **Suspicious Indicators**: Abnormal payloads, frequency spikes, unauthorized access, tunneling

---

## Use Cases

- Visual reference for SOC analysts
- Training material for cybersecurity teams
- Integration into playbooks or dashboards
- Mapping alerts to protocol behavior
