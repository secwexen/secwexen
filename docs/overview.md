# Overview

## Purpose

This repository is a personal knowledge base focused on cybersecurity research, ethical hacking, and technical reflections.  
It serves as a portfolio of applied security notes, blog posts, and personal insights.

## Structure

- **blog/** → Technical articles and writeups  
- **docs/** → Documentation and supporting notes  
- **portfolio/** → CTF writeups, labs, and project examples  
- **reflections/** → Personal thoughts and research reflections  
- **DISCLAIMER.md** → Legal disclaimer (educational use only)  
- **ETHICS.md** → Ethical guidelines for security research  
- **LICENSE** → Open source license  
- **README.md** → Main introduction to the repository  

## Usage

- Browse the **portfolio/** for practical examples.  
- Read **blog/** for detailed technical posts.  
- Check **reflections/** for personal commentary.  
- Refer to **docs/** for structured documentation.
