## Cybersecurity & Infosec Terms

- **CyberSecurity** – The practice of protecting systems, networks, and programs from digital attacks.
- **Ethical Hacking** – Authorized attempts to identify vulnerabilities in systems to improve security.
- **Infosec** – Information Security; protecting information from unauthorized access, use, disclosure, disruption, modification, or destruction.
- **NetSec** – Network Security; measures to protect the integrity, confidentiality, and availability of network and data.
- **LinuxSec** – Security practices and configurations specific to Linux systems.
- **CloudSec** – Security practices for cloud computing environments.
- **DevSecOps** – Integrating security practices into DevOps workflows and automation.
- **Forensics** – Digital forensics; investigation of digital devices to collect and analyze evidence.
- **Malware** – Malicious software designed to disrupt, damage, or gain unauthorized access to systems.
- **ZeroTrust** – A security model assuming no implicit trust; every access request must be verified.
- **SIEM** – Security Information and Event Management; tools for monitoring, detecting, and analyzing security events.
- **Red Team** – Security professionals simulating attacks to test defenses.
- **Blue Team** – Security professionals defending systems against attacks.
